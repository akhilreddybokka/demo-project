Branching in Git.
-------------

1. Head is a pointer for current branch.
2. git branch bug-fix - to create new branch for the copy of current branch we are in , 
    basically head point branch, but it doesnt switch to that branch after creating,
     we are still in the earlier branch.
3. git branch - to list all branches.
4. switch branches - git switch bug-fix
5. git switch -c patch -- to create branch and switch to new branch
